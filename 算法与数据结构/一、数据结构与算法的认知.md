## 2、算法与算法设计

### 2.1. 算法的定义

```python
# 对特定问题求解步骤的一种描述，是指令的有序序列

# 指令：是指能够被人或者是机器等计算装置执行。他可以是计算机指令，也可以是我们平时的语言文字
```

### 2.2. 算法的特性

```Python
1. 有穷性: 算法在执行有限的步骤之后，自动结束而不会出现无限循环，并且每一个步骤在可接受的时间内完成
2. 确定性: 算法的每一个步骤都具有确定的含义，不会出现二义性
3. 可行性: 算法的每一步都必须是可性的，也就是说能够执行有限的次数完成
4. 输入、输出:算法具有另个或多个输入、至少有一个输出
```

### 2.3 算法的设计要求

```python
# 时间复杂度
# 空间复杂度
```

#### 2.3.1 时间复杂度(重点)

```python
# 定义: 算法的执行时间随规模增长而增长的趋势 T(n) = O(f(n))
# 估算方法: 以算法中重复执行的次数作为算法时间复杂度的依据
```

#### 2.3.2 空间复杂度

```python
# 定义:算法执行过程中所需的最大空间 S(n) = O(f(n))
# 估算方法:输入数据所占空间 + 程序所占的空间 + 辅助变量所占的空间 
```
